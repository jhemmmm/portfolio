# Personal Portfolio

A professional personal portfolio website showcasing my skills in web development, backend engineering, game development, and reverse engineering. Built with Vue 3 and Tailwind CSS, deployed on Vercel.

---

## Features

- **Hero section** — Full-screen introduction with typewriter role animation and profile photo
- **About** — Bio, career timeline, quick stats, and resume download
- **Skills** — Categorized skill badges with filter tabs (Languages, Backend & Frameworks, Databases & Caching, Infrastructure & Tools)
- **Projects** — Filterable project grid with detail modal, screenshot lightbox, and live/GitHub links
- **Blog** — Articles with category filter, search, full post view with syntax-highlighted code blocks
- **Contact** — Contact form with validation and direct social/contact links
- **Dark / Light mode** — Persisted via localStorage, no flash on load
- **Fully responsive** — Mobile hamburger menu, adaptive layouts across all breakpoints
- **Page transitions** — Smooth fade animations between routes
- **Deployed on Vercel** — SPA routing handled via `vercel.json` rewrites

---

### Portfolio Stack

| Technology              | Purpose                 |
| ----------------------- | ----------------------- |
| Vue 3 (Composition API) | UI framework            |
| Vite                    | Build tool & dev server |
| Tailwind CSS v3         | Styling                 |
| Vue Router v4           | Client-side routing     |
| @vueuse/motion          | Scroll animations       |
| @tailwindcss/typography | Blog post prose styling |
| lucide-vue-next         | Icons                   |
| Vercel                  | Hosting & deployment    |

---

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
